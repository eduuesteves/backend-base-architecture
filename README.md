# 🚀 Backend Base Architecture

Projeto evolutivo focado em alta performance e boas práticas de mercado, construído com:
- Node.js & TypeScript
- Docker (Ambiente isolado de desenvolvimento)

## 🛠️ Como rodar o projeto
1. Certifique-se de ter o Docker e o Docker Compose instalados.
2. Execute o comando na raiz do projeto: `docker compose up --build`
3. A aplicação estará rodando com hot-reload ativo.
